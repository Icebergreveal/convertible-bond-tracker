你是金融公告结构化抽取专家。请从以下可转债公告内容中提取关键信息。

## 任务要求
- 只根据输入文本抽取字段，不编造信息
- 不确定或文本中不存在时输出 null
- 每个关键字段必须提供 evidence_text，必须是原文片段
- 日期格式统一为 YYYY-MM-DD
- 价格保留2位小数
- 输出必须是合法 JSON

## 公告类型
{{ann_type}}

## 输入文本
{{content}}

## 输出字段
如果是下修类公告，请提取：
- trigger_rule: 触发条件描述（原文）
- original_conv_price: 修正前转股价格（数字）
- new_conv_price: 修正后转股价格（数字）
- effective_date: 生效日期
- evidence_page: 页码（如果能找到）
- evidence_text: 支持以上字段的原文片段

如果是强赎类公告，请提取：
- redemption_trigger: 赎回触发条件描述（原文）
- redemption_price: 赎回价格（数字）
- record_date: 登记日期
- last_convert_date: 最后转股日期
- evidence_page: 页码（如果能找到）
- evidence_text: 支持以上字段的原文片段

## 输出格式
{
  "trigger_rule": "原文内容",
  "original_conv_price": 数字或null,
  "new_conv_price": 数字或null,
  "effective_date": "YYYY-MM-DD或null",
  "redemption_trigger": "原文内容或null",
  "redemption_price": 数字或null,
  "record_date": "YYYY-MM-DD或null",
  "last_convert_date": "YYYY-MM-DD或null",
  "evidence_page": 数字或null,
  "evidence_text": "原文片段"
}
