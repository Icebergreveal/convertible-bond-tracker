# Crawl Spec

## 项目名称

可转债转股价下修与提前强赎双事件全生命周期跟踪及结构化分析

## 数据来源

巨潮资讯网公开公告。

## 抓取范围

- 股票池或行业：全部A股可转债
- 时间范围：2023-01-01 至 2026-12-31
- 市场：深交所（sz）、上交所（sh）
- 公告类型：下修相关、强赎相关
- 关键词：转股价向下修正、转股价格修正、转债赎回、提前赎回、强赎、触发转股价格向下修正、行使提前赎回权

## 数据量目标

- 目标：150+ PDF公告
- 最小要求：100+ PDF公告

## metadata 字段

| 字段 | 类型 | 说明 |
|---|---|---|
| doc_id | 字符串 | 唯一标识，由股票代码+日期+标题哈希组成 |
| stock_code | 字符串 | 正股代码 |
| stock_name | 字符串 | 正股名称 |
| bond_code | 字符串 | 转债代码 |
| bond_name | 字符串 | 转债名称 |
| ann_type | 字符串 | 公告类型分类 |
| publish_date | 日期 | 公告发布日期 |
| announcement_url | URL | 公告详情页链接 |
| pdf_url | URL | PDF下载链接 |
| download_status | 枚举 | success/failed/skipped |
| crawl_time | 日期时间 | 抓取时间 |
| notes | 字符串 | 备注信息 |

## PDF 保存规则

- 保存路径：data/pdf/{doc_id}.pdf
- 文件命名：{doc_id}.pdf
- 已存在文件：跳过下载，标记为skipped

## 限速与失败处理

- 请求间隔：1.5秒
- 重试次数：3次
- 失败记录：保存到 outputs/logs/failed_downloads.csv
- 错误类型：网络错误、超时、文件不存在、权限问题

## 合规说明

- 不绕过登录、验证码或访问限制
- 保留完整抓取日志
- 只抓取公开可访问数据
- 遵守robots.txt协议
